# cf_ai_oregontrail_

To view, visit https://oregon-trail-ai.tai-karir.workers.dev/ in browser.

Oregon Trail AI is a reimagining of the classic Oregon Trail game, powered by Cloudflare. It uses Workers AI (Llama 3.1), Durable Objects for memory, and a lightweight HTML frontend to let players travel west, make decisions, and survive the journey.

- AI Narration for dynamic storytelling
- Persistent state for tracking resources like food, health, and distance
- Players type choices to progress.
